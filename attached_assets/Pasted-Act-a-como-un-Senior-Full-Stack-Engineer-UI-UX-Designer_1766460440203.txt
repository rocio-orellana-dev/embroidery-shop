- /app/shop/page.tsx debe ser SERVER COMPONENT (async) y NO debe tener onChange ni handlers.
- La parte interactiva de filtros (selects/search) debe ser un CLIENT COMPONENT separado:
  - /components/storefront/ShopFilters.client.tsx
  - Este componente lee y escribe query params con useRouter/useSearchParams.
  - Para cambios, debe hacer router.push() con los params.
- En server page solo renderiza <ShopFilters ...props> como componente client con props serializables.
